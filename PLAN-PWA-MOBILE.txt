DETECCION MOBILE Y REDIRECCION
================================================================================

En index.html (desktop), agregar script que:
1. Detecta si es mobile (user agent o screen width < 768px)
2. Si es mobile â†’ redirige a /mobile/index-mobile.html
3. Si es desktop â†’ queda en index.html normal


================================================================================
ESTRUCTURA HTML MOBILE (TEMPLATE BASE)
================================================================================

Cada HTML mobile sigue esta estructura:

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="theme-color" content="#ef4444">
    <link rel="manifest" href="/manifest.json">
    <title>Excentrica</title>

    <!-- CSS Base (reutilizado) -->
    <link rel="stylesheet" href="/css/main.css">

    <!-- CSS Mobile (nuevos) -->
    <link rel="stylesheet" href="/css/base-mobile.css">
    <link rel="stylesheet" href="/css/layout-mobile.css">
    <link rel="stylesheet" href="/css/components-mobile.css">
</head>
<body class="mobile-app">

    <!-- Banner instalacion PWA -->
    <div id="install-banner"></div>

    <!-- Header mobile -->
    <header id="mobile-header"></header>

    <!-- Contenido de la pagina -->
    <main id="mobile-main">
        <!-- Contenido especifico aqui -->
    </main>

    <!-- Bottom navigation -->
    <nav id="mobile-bottom-nav"></nav>

    <!-- Scripts reutilizados -->
    <script src="/js/config.js"></script>
    <script src="/js/api.js"></script>
    <script src="/js/auth.js"></script>
    <script src="/js/utils.js"></script>

    <!-- Scripts mobile nuevos -->
    <script src="/js/pwa-mobile.js"></script>
    <script src="/js/nav-mobile.js"></script>
    <script src="/js/ui-mobile.js"></script>
    <script src="/js/app-mobile.js"></script>

    <script>
        MobileApp.init({ page: 'home' });
    </script>
</body>
</html>
```


================================================================================
ORDEN DE TRABAJO (FASES)
================================================================================

FASE 1 - Base PWA (~30 min)
---------------------------
1. manifest.json
2. sw.js
3. Iconos placeholder (podemos mejorar despues)

FASE 2 - Estructura Mobile (~1 hora)
------------------------------------
4. css/base-mobile.css (variables, reset)
5. css/layout-mobile.css (header, bottom nav, main)
6. js/pwa-mobile.js (install banner, offline)
7. js/nav-mobile.js (renderiza header y bottom nav)
8. js/app-mobile.js (init, estado global)

FASE 3 - Paginas Core (~1 hora)
-------------------------------
9.  mobile/index-mobile.html (home feed)
10. mobile/categorias-mobile.html (menu secciones)
11. mobile/perfil-mobile.html (usuario)
12. js/feed-mobile.js (logica del feed)
13. js/ui-mobile.js (cards, componentes)
14. css/components-mobile.css (cards, botones)

FASE 4 - Periodista (~30 min)
-----------------------------
15. mobile/publicar-mobile.html (form noticia)
16. Logica especifica en app-mobile.js

FASE 5 - Secciones (~2 horas)
-----------------------------
17-27. Resto de secciones mobile (noticias-mobile, mercaderia-mobile, etc)
28. css/pages-mobile.css (estilos especificos)

FASE 6 - Integracion (~30 min)
------------------------------
29. Redireccion mobile en index.html desktop
30. Testing y ajustes
